# DataFace

##Installation :

  Clone the project
  `git clone git@github.com:SteveCarrasco/dataface-front.git`

  Open the folder
  `cd dataface-front`

  Gulp installatiion:
  `npm install gulp -g`

  Install node modules:
  `npm install`

  Run project:
  `gulp`

  Open the following link in your browser:
  `http://localhost:3000`
